💬 WhatsApp
⚡ Collez ou tapez une liste de contacts — un par ligne.
Format accepté : Nom ; Téléphone ; Secteur ; Logiciel (séparateur ; ou , ou tab)
📱 Comment exporter vos contacts Android :
1. Ouvrir Contacts → Menu ⋮ → Gérer les contacts
2. Exporter → choisir Stockage interne → fichier .vcf
3. Transférer le fichier sur votre PC et l'importer ici

📱 iPhone : Paramètres → Contacts → Exporter au format vCard
📊 Exporter depuis Google Contacts :
1. Ouvrir contacts.google.com
2. Cocher les contacts → Exporter → format Google CSV
3. Uploader le fichier .csv ici

Colonnes reconnues : Name, Phone 1 - Value, Organization 1 - Name, Email 1 - Value
💬 3 méthodes pour importer depuis WhatsApp :
A · Coller un numéro WhatsApp directement
B · Coller une liste de numéros (depuis carnet)
Un numéro par ligne, avec ou sans nom. Format: Nom : +225XXXXXXXXX ou juste +225XXXXXXXXX
C · Depuis WhatsApp → Partager contact
Dans WhatsApp : ouvrez un chat → appuyez sur le nom → Partager → sélectionnez SANIX CRM (si installé en PWA)
Ou : exportez la fiche contact → envoyez-vous le fichier .vcf → importez via l'onglet vCard
